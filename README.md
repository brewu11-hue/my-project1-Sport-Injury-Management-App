Sport Injury & Management App
Overview
The Sport Injury & Management App is a web/mobile application designed to help athletes, coaches, and medical staff track, manage, and recover from sports-related injuries. Built using Firebase Studio, the app provides secure data storage, real-time updates, and a user-friendly interface for monitoring recovery progress.

Features
User Authentication: Secure login and registration with Firebase Authentication.

Injury Logging: Record details of injuries (type, severity, date, treatment plan).

Recovery Tracking: Monitor rehabilitation progress with milestones and notes.

Notifications: Reminders for physiotherapy sessions or medical check-ups.

Analytics Dashboard: Visual insights into injury trends and recovery timelines.

Responsive Design: Optimized for both web and mobile devices.

Tech Stack
Frontend: HTML, CSS, JavaScript (responsive design principles)

Backend/Database: Firebase Realtime Database & Firestore

Authentication: Firebase Authentication

Hosting: vercel
